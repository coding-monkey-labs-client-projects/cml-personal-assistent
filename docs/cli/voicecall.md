# `cml-hive-assist voicecall`

`voicecall` is a plugin-provided command. It only appears if the voice-call plugin is installed and enabled.

Primary doc:

- Voice-call plugin: [Voice Call](/plugins/voice-call)

## Common commands

```bash
cml-hive-assist voicecall status --call-id <id>
cml-hive-assist voicecall call --to "+15555550123" --message "Hello" --mode notify
cml-hive-assist voicecall continue --call-id <id> --message "Any questions?"
cml-hive-assist voicecall end --call-id <id>
```

## Exposing webhooks (Tailscale)

```bash
cml-hive-assist voicecall expose --mode serve
cml-hive-assist voicecall expose --mode funnel
cml-hive-assist voicecall unexpose
```

Security note: only expose the webhook endpoint to networks you trust. Prefer Tailscale Serve over Funnel when possible.
