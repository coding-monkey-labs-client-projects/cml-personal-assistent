# `cml-hive-assist pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
cml-hive-assist pairing list whatsapp
cml-hive-assist pairing approve whatsapp <code> --notify
```
